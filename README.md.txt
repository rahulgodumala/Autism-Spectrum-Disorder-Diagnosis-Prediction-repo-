# Autism Spectrum Disorder Diagnosis Prediction

This project predicts Autism Spectrum Disorder (ASD) in toddlers using machine learning techniques like Random Forest and SVM. It includes a simple web interface for users to input screening data and get predictions.

## 📌 Project Highlights

- Data cleaning & preprocessing
- Feature selection
- Random Forest & SVM model training
- Flask-based web app with input form
- Model deployment using `.pkl` files
